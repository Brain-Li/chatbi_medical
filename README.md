
# 智能问数-医疗

This is a Vite + React prototype for a medical ChatBI / intelligent analytics workspace. The original Figma project is available at https://www.figma.com/design/kLcN1pRt0NBE1WkqSx4X62/%E6%99%BA%E8%83%BD%E9%97%AE%E6%95%B0-%E5%8C%BB%E7%96%97.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` to create a production build.

## Encoding

Source files are UTF-8. If Chinese text looks garbled in Windows PowerShell output, switch the console to UTF-8 before inspecting files:

```powershell
chcp 65001
$OutputEncoding = [System.Text.UTF8Encoding]::new()
```
